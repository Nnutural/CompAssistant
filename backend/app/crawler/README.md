# AI Crawler Scaffold

This directory is a placeholder scaffold for a future AI crawler capability.

Current scope:

- it is not wired into the existing agent runtime
- it is not exposed through any API route
- it does not perform real web crawling, login, scheduling, or storage

Why it exists:

- to reserve a clear backend boundary for future crawler work
- to keep source adapters, extraction pipelines, normalization, and storage
  bridges out of the current competition assistant runtime until they are
  intentionally designed

Possible future extension directions:

- source adapters
- extraction pipeline
- normalization layer
- storage bridge
